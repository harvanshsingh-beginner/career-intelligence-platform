# AI-Powered Interview & Career Intelligence Platform

An advanced AI-driven platform that helps users:

- Analyze resumes
- Calculate ATS scores
- Detect technical skills
- Recommend careers
- Recommend jobs
- Conduct AI mock interviews
- Evaluate interview answers
- Generate AI reports
- Track performance analytics

# Installation

## Backend Setup

```bash
cd backend

pip install -r requirements.txt

python app.py
```

## Frontend Setup

```bash
cd frontend

npm install

npm start
```
